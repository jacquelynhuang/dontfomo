BD Update 應用

把 SMT Project Update 變成你的週計劃

每週 SMT 出的 project update 內容密集、格式複雜。以下是把這些 update 轉換成實際行動的 Claude 應用場景。

每週必做
Chat Update 消化器 — 把 SMT update 貼給 Claude，立刻產出「我這週要做什麼」行動清單，含截止日期
Chat 賣家通知訊息 — 輸入活動細節，自動生成各活動的 LINE 推廣話術，長度控制在 200 字以內
Cowork NC% 週報分析 — 上傳 Dashboard 下載的追蹤數據，自動識別本週問題商品，輸出賣家介入優先清單
KPI 追蹤
Artifact NC% 試算器 — 輸入 Baseline / Target / 實際 NC%，即時計算達成率和 KPI 得分（10分制）
Artifact FBS 開通率試算 — 輸入賣家商品數和已開通數，自動計算距 30% 門檻還差多少，以及可獲得哪些獎勵
Cowork 月底衝刺排序 — 上傳賣家名單和 KPI 狀況，智能排出今天最應該打電話的順序和溝通重點
本週 SMT 重點對應
FBS 6月獎勵 — 讓 Claude 幫你整理每個賣家符合哪個方案（A/B/C/D），客製化推廣話術
NC% Q3 KPI Pool 調整 — 讓 Claude 分析你的 tracking pool，找出高影響力品項優先確認
H2 降% 專案 — 貼入活動說明，讓 Claude 幫你準備賣家版說明話術和 FAQ
6.18 當日出貨 / 週末出貨 — 生成參與賣家的追蹤清單格式，每天快速更新出貨達標狀況
